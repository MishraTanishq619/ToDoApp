# Flask Todo App

-   Author : Tanishq Mishra
-   Location: Chattisgarh, India

This is a ToDo App , made on Micro-Backend Framework called Flask.
This requires Python interpreter to run.

Steps to run:

1. First, Go to folder "backend/flask/Tanishq-08-Rajnandgaon/src", Open Terminal here.

2. Now, Install Flask:

```bash
pip install flask
pip install flask_sqlalchemy
```

3. Then, run the server:

```bash
python app.py
 or
python3 app.py
```

Open [http://localhost:5000](http://localhost:5000) with your browser to see the result.
